üíé Core Mechanics (The "Fresh" Loop)

1. The Vibeprint

Instead of a bio, users build a sonic and aesthetic fingerprint.

  • Spotify/Apple Music API integration (Non-negotiable)
  • "Vibe Tags" (e.g., #3AM_Coding, #Techno_Bunker, #Matcha_Lover)
  • Constraint: No text bios allowed. Only tags and media.

2. Blind Matching

Users are matched purely on Vibeprint overlap + Location.

3. The "Defrost" Chat

Chat starts frozen (blurred/anonymous). As engagement increases (messages exchanged, reaction speed), the profile slowly "defrosts" or reveals details.
